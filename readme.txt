# TSM v7.1 UFO Model
This model implements the Topological Substructure Model (TSM) for use in MadGraph5.

## Usage
1. Download this folder.
2. Place it in the `models/` directory of your MadGraph5 installation.
3. Run MadGraph and type: `import model TSM_v7_1_UFO`
4. Generate signal: `generate t > w+ b x`
